A012025

Broken Access Control ↑ #1

100% de apps testadas presentan alguna falla · Mayor nº de CVEs

IDOR (Insecure Direct Object Reference): al cambiar el parámetro ?id= en la URL se accede a perfiles, facturas o datos de otros usuarios sin ningún control.

⚠ Vulnerable:3001 ✓ Segura:3002
A022025

Security Misconfiguration ↑ de A05→A02

100% de apps con alguna mala configuración · Sube 3 posiciones

Endpoint /debug expuesto con env vars y contraseñas, stack traces con configuración interna, panel admin sin autenticación y CORS demasiado permisivo.

⚠ Vulnerable:3003 ✓ Segura:3004
A032025

Software Supply Chain Failures ✦ NUEVO

#1 en la encuesta comunitaria · SolarWinds, Log4Shell, Bybit 2025

Prototype Pollution via librería de merge desactualizada (simula CVE-2019-10744 lodash). Un payload {"__proto__":{"isAdmin":true}} contamina Object.prototype en el servidor y escala privilegios.

⚠ Vulnerable:3005 ✓ Segura:3006
A042025

Cryptographic Failures ↓ de A02→A04

Criptografía débil o ausente · Datos sensibles expuestos

Contraseñas almacenadas con MD5 sin sal: crackeables en <1ms con rainbow tables. La versión segura usa bcrypt con cost=12 — infactible crackear en tiempo razonable.

⚠ Vulnerable:3007 ✓ Segura:3008
A052025

Injection ↓ de A03→A05

100% de apps vulnerables · 14k+ CVEs de SQL Injection

SQL Injection: concatenación directa de input en la query. ' OR '1'='1' -- bypasea el login; UNION attack vuelca toda la base de datos. La versión segura usa consultas parametrizadas.

⚠ Vulnerable:3009 ✓ Segura:3010
